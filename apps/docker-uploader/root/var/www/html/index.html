About Docker Uploader

Docker Uploader

Version 5.0.0

Description

Docker Uploader is a powerful and efficient file upload management system designed to run in Docker containers. It provides a modern web interface for monitoring and controlling file uploads with features like bandwidth management, concurrent transfer control, and real-time progress tracking.

Key Features

  • Real-time upload monitoring and progress tracking
  • Bandwidth limiting and transfer management
  • Multiple concurrent upload support
  • Complete upload history with statistics
  • Pause and resume functionality
  • 17 beautiful theme options
  • Responsive design for mobile and desktop
  • Docker containerized deployment
